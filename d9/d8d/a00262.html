Detailed Description

Class representing the Activity for app settings.

It displays the app settings and enables customization for the users. There are two customizations that are present :

  1. 3D Mouse Sensitivity.
  2. 2D Mouse Sensitivity.

Each of these can have three possible values - : "Low", "Medium" and "High". The default value is "Medium".

This class is made separate from the MainActivity to enable the "Settings" to be opened from any fragment of the MainActivity. This avoids overlapping of other fragments - that are already rendered - by the SettingsFragment.

It also implements the SharedPreferences.OnSharedPreferenceListener of the Android API. The onSharedPreferenceChanged(SharedPreferences, String) method listens to any changes made in the app settings.
