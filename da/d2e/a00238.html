Detailed Description

Accesses the sensor hardware to read sensor data.

Classes implementing this interface provide an orientation of the device either by directly accessing hardware, using Android sensor fusion or fusing sensors itself. The orientation can be provided as a rotation matrix (project.android.sensor.representation.MatrixF4x4) or a quaternion (project.android.sensor.representation.Quaternion).
