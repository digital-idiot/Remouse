22  * <p>
23  * The constructor {@link #Client(String)} is used to set up a TCP connection
24  * with the respective server. The sensor data obtained through
25  * {@link project.android.KeyboardFragment}, {@link project.android.MouseFragment}
26  * and {@link project.android.TouchpadFragment} are wrapped using {@link DataWrapper}
27  * which internally uses the <code>com.google.Gson</code> API. The wrapped <i>JSON
28  * string</i> is encrypted using the
29  * {@link project.android.security.EKEProvider#encryptString(String)} method and
30  * transmitted over the network.
31  * </p>
