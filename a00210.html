Detailed Description

Encapsulates the data sent to the server during TCP communication.

This data contains all the information acquired from the project.android.MouseFragment, the project.android.TouchpadFragment and the project.android.KeyboardFragment. It contains a type of operation field which helps the server determine the operation requested by the client and a data field which acts as the operand for the input operations.

For keyboard input, the type of operation field contains Key and the data field contains the sequence of characters and keyboard strokes for input. In case of special keys, the data field contains the special key type.

For 3-D mouse movement operation, the type of operation field contains Mouse_Move and the data field contains a project.android.sensor.representation.Quaternion object.

For 2-D mouse movement operation, the type of operation field contains Mouse_Touch and the data field contains a pair of integers representing the displacement along the x-axis and y-axis relative to the current mouse pointer location.

For mouse button click operation, the type of operation field contains Mouse_Button and the data field contains any of the following :

  • left - representing left click.
  • right - representing right click.
  • middle - representing middle button click.
  • upscroll - representing scroll up.
  • downscroll - representing scroll down.
