Detailed Description

Extends the AsyncTask class of the Android API.

It performs the following operations :

  • Displaying a ProgressDialog.
  • Constructing a project.android.security.EKEProvider object that will be used for encryption and decryption of TCP packets.
  • Starting a ClientIOThread.

AsyncTask enables these operations to performed asynchronously by defining them in the onPreExecute(), doInBackground(ServerInfo...) and onPostExecute(EKEProvider) methods respectively. The starting of ClientIOThread waits for the construction of the project.android.security.EKEProvider object. A ProgressDialog is displayed for this duration.
