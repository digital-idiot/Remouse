Detailed Description

Manages the operations on the KeyStore.

This class provides a java.security.KeyStore which stores :

  • The private key (java.security.PrivateKey).
  • A self-signed certificate (java.security.cert.Certificate) that contains the public key and the digital signature.

The KeyStore is protected by a randomly generated password.
