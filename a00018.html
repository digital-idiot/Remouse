Detailed Description

Class representing the Fragment for providing the GUI frontend for the 3D mouse module.

The following features have been provided:

  • 3D mouse movement.
  • Single left click and double left click.
  • Right click.
  • Middle click.
  • Scrolling

The 3D mouse movement feature uses the gyroscope and accelerometer sensors. The project.android.sensor.orientation.OrientationProvider and project.android.sensor.orientation.KalmanFilterProvider classes are responsible for processing the sensor data. The clicks and scrolling feature uses buttons.

Note: This 3D mouse module does not work with Android devices where the mentioned sensors are not present.
