Detailed Description

Implementation of the java.lang.Runnable for the thread responsible for sending keyboard data to the server.

This thread maintains a buffer to store the data to be sent to the server. It is implemented as a java.util.concurrent.LinkedBlockingQueue. The data contains two fields :

  • A data field which represents the data to be sent, and
  • A boolean flag which determines if the data represents a special key.

The data is inserted to the buffer by the listener methods defined in the project.android.KeyboardFragment class. This thread continuously monitors the buffer to check if data is available. If available, it sends the data.

This thread is started when the keyboard UI is loaded and the project.android.KeyboardFragment#onCreateView(LayoutInflater, ViewGroup, Bundle) method is called. It is stopped by invoking the setStopFlag() method when the keyboard UI is destroyed and the project.android.KeyboardFragment#onDestroyView() method is called.
