Detailed Description

Class representing the client.

The constructor Client(String) is used to set up a TCP connection with the respective server. The sensor data obtained through project.android.KeyboardFragment, project.android.MouseFragment and project.android.TouchpadFragment are wrapped using DataWrapper which internally uses the com.google.Gson API. The wrapped JSON string is encrypted using the project.android.security.EKEProvider#encryptString(String) method and transmitted over the network.
